## Project Setup

### Install

```bash
$ yarn

$ npm install
```

### Development

```bash
$ yarn dev

$ npm run dev
```

### Build

```bash
# For windows
$ yarn build:win
or
$ npm run build:win

# For macOS
$ yarn build:mac
or
$ npm run build:mac

# For Linux
$ yarn build:linux
or
$ npm run build:linux
```
